BaseConfig é um pacote que facilita a configuração inicial de projetos, automatizando a instalação e configuração de ferramentas essenciais.

## Instalação

Você pode executar o pacote via npm:

```sh
npx @leandrobataglia/baseconfig
```

## Uso

O pacote fornece um CLI para configurar ferramentas conforme necessário.

### Opções Disponíveis

- `--help`: Exibe a lista de ferramentas disponíveis.
- `--version`: Mostra a versão do pacote.
- `--all`: Instala todas as ferramentas disponíveis.
- `[nomes das ferramentas]`: Instala apenas as ferramentas especificadas.

## Desenvolvimento

Clone o repositório e instale as dependências:

```sh
git clone https://github.com/leandrobataglia/baseconfig.git
cd baseconfig
npm install
```
